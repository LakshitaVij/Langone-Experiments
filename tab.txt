
## Overview
The tabular model is a HistGradientBoostingClassifier (sklearn) trained on 37 clinical and lesion-based features per patient.

HistGradientBoosting is used because:
- It handles missing values natively
- More powerful than logistic regression for tabular data
- Fast, no GPU needed

**Feature engineering:**
- Lesion slots: Takes top 3 by PIRADS descending
- Zone parsing: 4 categories (peripheral, transition, central, other) with double-counting for multi-zone lesions
  - Multi-zone lesions (e.g., "peripheral extending into transition") now captured via primary + secondary zone columns
  - Cleaner anatomy: dropped location transverse feature entirely

**Exclusion criteria (applied at cohort level):**
- Removed rows where has_hip_implant == True (82 rows)
- Removed rows where has_prior_pca_treatment == True (0 rows)

**Final dataset:**
- 8,254 labeled rows → 8,172 after exclusions (98.0%)
- csPCa balance: 50.9% positive, 49.1% negative

### Per-Lesion Features (24 total: 3 slots × 8 features/slot)

Primary features (5):
- `lesion_N_pirads`

- `lesion_N_zone_transition` (one-hot encoded)

- `lesion_N_zone_secondary_transition` (one-hot encoded)
- `lesion_N_zone_secondary_central` (one-hot encoded)

## Training

- **Algorithm:** HistGradientBoostingClassifier (max_iter=1000, early_stopping=True)
- **CV strategy:** 5-fold StratifiedGroupKFold grouped by PatientID
  - Grouped because 115 patients had rows spanning train/val/test; using raw split column would leak patient info across folds
- **Model output:** Saved as cspca_model_v4.pkl via joblib

## Zone Parsing Logic (v4)

The zone parser converts ~500 unique free-text radiologist zone descriptions into 4 mutually exclusive categories:

1. **Peripheral (PZ):** 67.6% of lesions
2. **Transition (TZ):** 21.5%
3. **Central (CZ):** 2.8%
4. **Other:** 3.0% (location-only descriptions like "apex", "seminal vesicle"; rare entire-gland cases)

For multi-zone lesions (e.g., "peripheral extending into transition"):
- Primary zone: first zone keyword found (captures dominant location)
- Secondary zone: second zone keyword if present (captures extension/involvement)
- Example: "peripheral with extension into transition zone" → primary=peripheral, secondary=transition

Entire-gland lesions ("entire prostate", "diffuse throughout") are assigned to "other" primary zone and flagged via secondary zones if applicable. These are rare (3 patients).
