The tabular model is a HistGradientBoostingClassifier (sklearn) trained on 17 clinical 
features per patient.

HistGradientBoosting is used because:
- it handles missing values itself 
- more powerful than logistic regression for tabular data 
- fast, no GPU needed


Features:
1. maxPIRADS
2. psa_log
3. psa_present
4. psa_density
5. prostate_volume_cc
6. prostate_volume_present
7. intra_vesical_protusion_cm
8. intra_vesical_protutsion_present
9. prior_prostate_procedure_clean
10. prior_prostate_procedure_present
11. has_prior_pca_treatment_clean
12. has_prior_pca_treatment_present
13. has_hip_implant_clean
14. has_hip_implant_present
15. lesion_count
16. lesion_max_size_mn
17. lesion_has_epe


Training:
- 5 fold StratifiedGroupKFold grouped by PatientID
- Grouped because 115 patients had rows spanning train/val/test,  using the raw split column would leak patient info across folds

Output:
- Saved as cspca_model.pkl via joblib
- Mean cross validated AUC : 0.8720
